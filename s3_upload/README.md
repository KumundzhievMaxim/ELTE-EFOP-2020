# S3 Data Uploader
The tool assumed to download data from TFL and upload to the dedicated S3 Bucket 

## Get help: 
````bash
python handler.py -h   
````

## Run the Script: 
````bash
python handler.py --id --key --year  
````
#### Example: 
````bash
python handler.py --id tfl_id --key tfl_key --year 2020  
````
