# Lambda Data Preprocessor
The tool assumed to preprocessor uploaded data in the dedicated S3 Bucket

#### Get help: 
````bash
cd lambda_transform && python handler.py -h   
````
#### Run the Script: 
````bash
cd lambda_transform && python handler.py   
````
#### Example: 
````bash
cd lambda_transform && python handler.py   
````
